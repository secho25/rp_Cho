Procedure

The step-by-step methodology used to conduct this study.


Step

Description

Image

1

Data Preprocessing

pixel quality assessments performed to detect and remove cloud cover, which prevents accurate index calculations; water pixels also masked to avoid skewing land surface temperature data

2

Hyperparameter Tuning

convolutional neural network (CNN) was trained on the processed data; hyperparameters such as learning rate, number of layers, and kernel size were fine-tuned for the highest possible accuracy

3

Model Evaluation

with optimized hyperparameters, model tested on an unseen test dataset to assess performance with mean squared error used as loss function

4

Graph Creation

various density scatter plots comparing actual vs. predicted land surface temperature values were created using the Seaborn Python library to provide a visual representation of the results

5

Statistical Analysis & Conclusions

values such as the 95% confidence interval, p-value, t-statistic, and Cohen's d were computed to measure the significance of the study's results; these values were used to draw conclusions about model performance & experiment as a whole
